# إدارة البودكاست

## الإضافة
1. افتح صفحة إضافة بودكاست
![لقطة إعدادات البودكاست](/images/manage-1.png)
2. أدخل معلومات البودكاست وارفع ملف الصوت والصورة، ثم انقر 【نشر】
![لقطة إعدادات البودكاست](/images/manage-2.png)

## التعديل
1. افتح صفحة تعديل بودكاست
![لقطة إعدادات البودكاست](/images/manage-3.png)
2. عدّل معلومات البودكاست ثم انقر 【حفظ】
![لقطة إعدادات البودكاست](/images/manage-4.png)

## مرجع الحقول

  | الحقل | الوصف |
  |---|---|
  | ملف الصوت <span style="color: red;">*</span> | مطلوب. ارفع ملفًا صوتيًا أو أدخل رابط ملف صوتي (https). |
  | المدة (بالثواني) | يُكتشف تلقائيًا عند الحفظ من “Audio File”، بالثواني، للقراءة فقط. |
  | طول الصوت (بالبايت) | يُكتشف تلقائيًا عند الحفظ من “Audio File”، للقراءة فقط. |
  | نوع MIME للصوت | يُكتشف تلقائيًا عند الحفظ من “Audio File”، للقراءة فقط، والافتراضي `audio/mpeg`. |
  | المحتوى الصريح <span style="color: red;">*</span> | مطلوب. اختر `clean` أو `explicit`. |
  | نوع الحلقة <span style="color: red;">*</span> | مطلوب. اختر `full` أو `trailer` أو `bonus`. |
  | رقم الحلقة | اختياري، لكن يُنصح به. رقم صحيح. |
  | رقم الموسم | اختياري. رقم صحيح. |
  | مؤلف الحلقة (تجاوز) | اختياري. يستبدل مؤلف القناة لهذه الحلقة فقط. |
  | غلاف الحلقة (مربع) | اختياري. صورة مربعة، يُنصح بـ 1400-3000px. تستبدل غلاف القناة. |
  | النص التفريغي (اختياري) | اختياري. ارفع ملف تفريغ أو أدخل رابطًا، مع دعم vtt/srt/txt/pdf وغيرها. |
  | عنوان iTunes (اختياري) | اختياري. قيمة بديلة لعنوان الحلقة في Apple Podcasts. |
  | الفصول (Podcasting 2.0) | اختياري. ملف JSON أو رابط للفصول. |
  | نوع MIME للفصول | اختياري. نوع MIME لملف الفصول، والافتراضي `application/json+chapters`. |
  | مقتطفات صوتية (Podcasting 2.0) | حقل مركب. أضف مقطعًا أو أكثر؛ كل مقطع يتضمن وقت البدء والمدة والعنوان. |
  | العنوان الفرعي | اختياري. عنوان فرعي قصير لـ iTunes. |
  | الملخص | اختياري. ملخص نصي لـ iTunes. |
  | GUID مخصص (اختياري) | اختياري. GUID مخصص لـ RSS؛ إذا كان فارغًا يُستخدم رابط الحلقة. |
  | حظر iTunes | اختياري. `yes` يخفي الحلقة في Apple Podcasts، والافتراضي `no`. |
  | الأعضاء | اختر أعضاء الحلقة، عادةً المدراء أو المؤلفين أو المحررين. |
  | الضيوف | اختر ضيوف الحلقة، عادةً المساهمين. |
