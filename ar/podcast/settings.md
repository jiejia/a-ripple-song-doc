# إعدادات البودكاست
تُستخدم هذه الحقول بشكل أساسي لإنشاء RSS الخاص بالبودكاست.

## كيفية الفتح
![لقطة شاشة لإعدادات البودكاست](/images/settings-1.png)

## مرجع الحقول
| الحقل | الوصف |
|---|---|
| رابط RSS للبودكاست | يعرض عنوان RSS الحالي للبودكاست للقراءة فقط. يمكنك النقر داخل الحقل لتحديده ونسخه، أو فتح الـ feed مباشرة. |
| عنوان البودكاست <span style="color: red;">*</span> | مطلوب. عنوان البودكاست. إذا كان فارغًا، فسيعود RSS إلى عنوان الموقع. |
| العنوان الفرعي للبودكاست | اختياري. عنوان فرعي قصير أو شعار يظهره بعض العملاء. |
| وصف البودكاست <span style="color: red;">*</span> | مطلوب. وصف نصي للبودكاست. يُرسل إلى RSS كـ description / ملخص iTunes. |
| مؤلف البودكاست (itunes:author) <span style="color: red;">*</span> | مطلوب. مؤلف أو منتج البودكاست الذي يظهر في الأدلة. |
| اسم المالك <span style="color: red;">*</span> | مطلوب. قيمة `<itunes:owner><itunes:name>` لمعلومات المالك. |
| بريد المالك الإلكتروني <span style="color: red;">*</span> | مطلوب. قيمة `<itunes:owner><itunes:email>`. استخدم عنوانًا يمكنه استقبال رسائل التحقق. |
| غلاف البودكاست (1400-3000 بكسل مربع) <span style="color: red;">*</span> | مطلوب. صورة غلاف البودكاست. يجب أن تكون JPG/PNG، مربعة، 1400-3000px، وأقل من 512KB. |
| العلامة الافتراضية للمحتوى الصريح <span style="color: red;">*</span> | مطلوب. مؤشر المحتوى الصريح الافتراضي. اختر clean أو explicit. يمكن لحقول الحلقة أن تتجاوزه. |
| اللغة (RFC 5646) <span style="color: red;">*</span> | مطلوب. لغة البودكاست، مثل en-US أو zh-CN أو ja-JP. |
| الفئة الرئيسية (Apple Podcasts) <span style="color: red;">*</span> | مطلوب. الفئة الرئيسية في Apple Podcasts. يجب اختيار فئة واحدة على الأقل. |
| الفئة الثانوية (اختياري) | اختياري. فئة ثانوية تدعمها بعض أدلة البودكاست. |
| حقوق النشر (اختياري) | اختياري. بيان حقوق النشر يُرسل إلى `<copyright>`. |
| نوع iTunes (itunes:type) | اختياري. نوع العرض في Apple Podcasts. الخيارات: غير محدد، episodic، serial. |
| عنوان iTunes (اختياري) | اختياري. عنوان مخصص لـ Apple Podcasts. استخدمه فقط إذا كان يجب أن يختلف عن عنوان البودكاست العادي. |
| حظر iTunes (itunes:block) | اختياري. yes يخفي البودكاست بالكامل في Apple Podcasts. القيمة الافتراضية no. |
| اكتمال iTunes (itunes:complete) | اختياري. yes يعني أن البودكاست مكتمل ولن يتم تحديثه. القيمة الافتراضية no. |
| رابط RSS جديد لـ iTunes (itunes:new-feed-url) | اختياري. استخدمه فقط عند الانتقال إلى عنوان RSS جديد. |
| podcast:locked | حقل Podcasting 2.0. القيمة الافتراضية yes لمنع نقل feed غير المصرح به. |
| مالك podcast:locked (اختياري) | اختياري. بريد المالك المستخدم للتحقق من نقل feed في Podcasting 2.0. |
| podcast:guid (اختياري) | اختياري. GUID لـ Podcasting 2.0. إذا كان فارغًا، سيستخدم feed عنوان الموقع كخيار احتياطي. |
| رمز التحقق من Apple Podcasts | اختياري. رمز التحقق لـ Apple Podcasts، ويُخرج كـ `podcast:txt purpose="applepodcastsverify"`. |
| روابط تمويل Podcasting 2.0 | حقل مركب. يمكنك إضافة عدة روابط للتمويل أو الدعم؛ كل عنصر يحتوي على URL و Label. إذا كان فارغًا، فلن يُخرج `podcast:funding`. |
| المولد (اختياري) | اختياري. محتوى `<generator>` الخاص بـ RSS. إذا كان فارغًا، فلن تُخرج العلامة. |
