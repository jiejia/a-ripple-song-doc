# শুরু করুন

## ইনস্টলেশন
### WordPress প্লাগইন ডিরেক্টরি থেকে
1. WordPress অ্যাডমিনে প্লাগইনস পেজ খুলুন
2. সার্চ বক্সে A Ripple Song লিখে Enter চাপুন
3. A Ripple Song প্লাগইন দেখালে 【এখনই ইনস্টল করুন】 ক্লিক করুন
4. ইনস্টল শেষ হলে 【সক্রিয় করুন】 ক্লিক করুন

### প্লাগইন প্যাকেজ থেকে
1. [Github Release](https://github.com/jiejia/a-ripple-song/releases) থেকে `a-ripple-song-podcast.zip`-এর সর্বশেষ সংস্করণ ডাউনলোড করুন, অথবা [WordPress Themes](https://wordpress.org/plugins/a-ripple-song/) থেকে নিন
2. WordPress-এ 【নতুন প্লাগইন যোগ করুন】 পেজ খুলুন, 【প্লাগইন আপলোড করুন】 ক্লিক করুন, ফাইল বাছাই করে 【এখনই ইনস্টল করুন】 চাপুন
3. 【সক্রিয় করুন】 ক্লিক করুন

### কমান্ড লাইন থেকে
1. প্রথমে দেখুন wp cli ইনস্টল আছে কি না; না থাকলে https://wp-cli.org/ থেকে ইনস্টল করুন
2. WordPress root-এ এই কমান্ড চালান: `wp plugin install a-ripple-song`
