# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song) একটি ক্লাসিক WordPress থিম। [পডকাস্ট প্লাগইন](https://github.com/jiejia/a-ripple-song) ইনস্টল করার পর অডিও প্লেয়ার ও পডকাস্ট widget ব্যবহার করা যায়।

## ইনস্টলেশন

### WordPress থিম ডিরেক্টরি থেকে
1. WordPress অ্যাডমিনে 【নতুন থিম যোগ করুন】 পেজ খুলুন
2. সার্চ বক্সে Daisy A Ripple Song লিখে Enter চাপুন
3. Daisy A Ripple Song থিম দেখালে 【ইনস্টল】 ক্লিক করুন
4. ইনস্টল শেষ হলে 【সক্রিয় করুন】 ক্লিক করুন

### থিম প্যাকেজ থেকে
1. [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases) থেকে `daisy-a-ripple-song-podcast.zip`-এর সর্বশেষ সংস্করণ ডাউনলোড করুন, অথবা [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/) থেকে নিন
2. WordPress-এ 【থিম যোগ করুন】 পেজ খুলুন, 【থিম আপলোড করুন】 ক্লিক করুন, ফাইল বাছাই করে 【এখনই ইনস্টল করুন】 চাপুন
3. 【সক্রিয় করুন】 ক্লিক করুন

### কমান্ড লাইন থেকে
1. প্রথমে দেখুন wp cli ইনস্টল আছে কি না; না থাকলে https://wp-cli.org/ থেকে ইনস্টল করুন
2. WordPress root-এ `wp theme install daisy-a-ripple-song` চালান
