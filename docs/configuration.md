# 配置

建议按“先站点级配置 → 再播客级配置 → 最后播放器与小部件”的顺序设置。

## 站点级设置（占位）

- 站点 Logo / Favicon
- 全局配色方案（浅色/深色）
- 版权信息与页脚链接

## 播客级设置（占位）

- 播客名称 / 作者 / 简介
- 分类与关键词
- 默认封面
- 订阅链接（Apple Podcasts/Spotify/小宇宙等）

## 播放器设置（占位）

- 默认播放速度：1.0x
- 进度记忆：开启
- 波形/频谱：开启（如有）

## 推荐配置清单

- 首次上线前：完善 RSS 必填字段
- 上线后：开启统计与缓存策略
