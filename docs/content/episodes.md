# 剧集管理

本页覆盖从“写标题”到“发布到平台”的完整流程（占位）。

## 剧集结构建议

- 标题：`EPxx + 主题`（示例：EP01 你好，世界）
- 简介：开头 1 段总结 + 3–5 条要点
- 章节（可选）：按时间点列出（Podcasting 2.0 Chapters）
- Transcript（可选）：提供全文或摘要

## 元数据（占位）

- 时长：自动提取或手动填写
- 文件大小：自动提取（用于部分平台展示）
- 显式内容标记：Explicit / Clean

## 发布清单

- 音频可被公网访问（HTTPS，支持 Range 请求）
- 封面尺寸与格式符合平台要求
- 试听与移动端播放正常
