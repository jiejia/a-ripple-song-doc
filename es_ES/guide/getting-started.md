# Primeros pasos

## Instalación
### Desde el repositorio de plugins de WordPress
1. Abre la página de plugins en el administrador de WordPress
2. Escribe A Ripple Song en el buscador y pulsa Enter
3. Cuando aparezca el plugin A Ripple Song, haz clic en 【Instalar ahora】
4. Cuando termine la instalación, haz clic en 【Activar】

### Desde un paquete del plugin
1. Descarga la última versión de `a-ripple-song-podcast.zip` desde [Github Release](https://github.com/jiejia/a-ripple-song/releases), o desde [WordPress Themes](https://wordpress.org/plugins/a-ripple-song/)
2. Abre la página 【Añadir plugin】 en WordPress, haz clic en 【Subir plugin】, elige el archivo y pulsa 【Instalar ahora】
3. Haz clic en 【Activar】

### Instalación por línea de comandos
1. Primero comprueba si wp cli está instalado; si no, instálalo desde https://wp-cli.org/
2. En la raíz de WordPress, ejecuta `wp plugin install a-ripple-song`
