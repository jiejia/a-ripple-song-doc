# ¿Qué es A Ripple Song?

A Ripple Song es un plugin de podcast para WordPress y el núcleo de todo el sistema. Gestiona la estructura de contenido, la administración, el tratamiento de metadatos de audio y la publicación RSS para ofrecer a WordPress una capacidad completa de publicación de podcasts.

## Capacidades principales

- Añade tipos de contenido y categorías específicas para podcasts en WordPress
- Ofrece campos y ajustes de administración dedicados
- Detecta automáticamente la duración, el tamaño y el formato del audio
- Genera un RSS conforme a iTunes y Podcasting 2.0 listo para distribuirse en las principales plataformas
- Incluye campos de estadísticas

## Relación con el tema

El plugin A Ripple Song es el núcleo. El tema es la interfaz de presentación.

- El plugin se encarga de las funciones, los datos y la lógica de negocio
- El tema se encarga de mostrar las páginas y la experiencia de usuario

Sin el tema, el plugin sigue siendo el núcleo del sistema de podcast. El tema simplemente presenta ese contenido de una forma más completa y agradable de navegar.

## Resumen

A Ripple Song es el plugin central que da a WordPress capacidades completas de podcasting. El tema convierte ese contenido en una interfaz de sitio realmente utilizable.
