# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song) es un tema clásico de WordPress. Al instalar el [plugin de podcast](https://github.com/jiejia/a-ripple-song), añade reproductor de audio y widgets de podcast.

## Instalación

### Desde el repositorio de temas de WordPress
1. Abre la página 【Añadir tema】 en el administrador de WordPress
2. Escribe Daisy A Ripple Song en el buscador y pulsa Enter
3. Cuando aparezca el tema Daisy A Ripple Song, haz clic en 【Instalar】
4. Cuando termine la instalación, haz clic en 【Activar】

### Desde un paquete del tema
1. Descarga la última versión de `daisy-a-ripple-song-podcast.zip` desde [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases), o desde [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/)
2. Abre la página 【Añadir tema】, haz clic en 【Subir tema】, elige el archivo y pulsa 【Instalar ahora】
3. Haz clic en 【Activar】

### Instalación por línea de comandos
1. Primero comprueba si wp cli está instalado; si no, instálalo desde https://wp-cli.org/
2. En la raíz de WordPress, ejecuta `wp theme install daisy-a-ripple-song`

## Características

- 56 combinaciones de color elegantes
- Soporte de internacionalización
- Seguimiento de datos
- Transiciones de página sin recarga basadas en Swup
- Varios widgets adaptados al estilo del tema
- Reproductor de audio inmersivo

## Stack técnico
- [Vite](https://github.com/vitejs/vite) - herramienta de desarrollo y compilación front-end
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) - framework CSS utilitario
- [daisyUI](https://github.com/saadeghi/daisyui) - biblioteca de componentes UI basada en Tailwind CSS
- [Alpine.js](https://github.com/alpinejs/alpine) - framework ligero de interacción front-end
- [Lucide](https://github.com/lucide-icons/lucide) - biblioteca de iconos open source
- [Simple Icons](https://github.com/simple-icons/simple-icons) - biblioteca de iconos de marcas
- [Swup](https://github.com/swup/swup) - biblioteca de navegación sin recarga
- [Howler.js](https://github.com/goldfire/howler.js) - biblioteca de reproducción de audio
- [audioMotion-analyzer](https://github.com/hvianna/audioMotion-analyzer) - biblioteca de visualización espectral de audio
- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - framework de campos personalizados para WordPress
- [PHP-Scoper](https://github.com/humbug/php-scoper) - herramienta de aislamiento de namespaces PHP
