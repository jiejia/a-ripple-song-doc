# Premiers pas

## Installation
### Depuis le répertoire des plugins WordPress
1. Ouvrez la page des extensions dans l’administration WordPress
2. Saisissez A Ripple Song dans la recherche puis appuyez sur Entrée
3. Quand le plugin A Ripple Song apparaît, cliquez sur 【Installer maintenant】
4. Une fois l’installation terminée, cliquez sur 【Activer】

### Depuis une archive du plugin
1. Téléchargez la dernière version de `a-ripple-song-podcast.zip` depuis [Github Release](https://github.com/jiejia/a-ripple-song/releases), ou depuis [WordPress Themes](https://wordpress.org/plugins/a-ripple-song/)
2. Ouvrez la page 【Ajouter une extension】 dans WordPress, cliquez sur 【Téléverser une extension】, choisissez le fichier puis cliquez sur 【Installer maintenant】
3. Cliquez sur 【Activer】

### Installation en ligne de commande
1. Vérifiez d’abord que wp cli est installé ; sinon installez-le depuis https://wp-cli.org/
2. Dans la racine WordPress, exécutez `wp plugin install a-ripple-song`
