# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song) est un thème WordPress classique. Après installation du [plugin podcast](https://github.com/jiejia/a-ripple-song), il ajoute un lecteur audio et des widgets podcast.

## Démo
[https://podcast.aripplesong.me/](https://podcast.aripplesong.me/)

## Installation

### Depuis le répertoire des thèmes WordPress
1. Ouvrez la page 【Ajouter un thème】 dans l’administration WordPress
2. Saisissez Daisy A Ripple Song dans la recherche puis appuyez sur Entrée
3. Quand le thème Daisy A Ripple Song apparaît, cliquez sur 【Installer】
4. Une fois l’installation terminée, cliquez sur 【Activer】

### Depuis une archive du thème
1. Téléchargez la dernière version de `daisy-a-ripple-song-podcast.zip` depuis [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases), ou depuis [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/)
2. Ouvrez la page 【Ajouter un thème】, cliquez sur 【Téléverser un thème】, choisissez le fichier puis cliquez sur 【Installer maintenant】
3. Cliquez sur 【Activer】

### Installation en ligne de commande
1. Vérifiez d’abord que wp cli est installé ; sinon installez-le depuis https://wp-cli.org/
2. Dans la racine WordPress, exécutez `wp theme install daisy-a-ripple-song`

## Fonctionnalités

- 56 palettes de couleurs élégantes
- Support de l’internationalisation
- Suivi de données
- Transitions de page sans rafraîchissement basées sur Swup
- Plusieurs widgets adaptés au style du thème
- Lecteur audio immersif

## Stack technique
- [Vite](https://github.com/vitejs/vite) - outil de développement et de build front-end
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) - framework CSS utilitaire
- [daisyUI](https://github.com/saadeghi/daisyui) - bibliothèque de composants UI basée sur Tailwind CSS
- [Alpine.js](https://github.com/alpinejs/alpine) - framework léger d’interaction front-end
- [Lucide](https://github.com/lucide-icons/lucide) - bibliothèque d’icônes open source
- [Simple Icons](https://github.com/simple-icons/simple-icons) - bibliothèque d’icônes de marques
- [Swup](https://github.com/swup/swup) - bibliothèque de navigation sans rechargement
- [Howler.js](https://github.com/goldfire/howler.js) - bibliothèque de lecture audio
- [audioMotion-analyzer](https://github.com/hvianna/audioMotion-analyzer) - bibliothèque de visualisation spectrale audio
- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - framework de champs personnalisés WordPress
- [PHP-Scoper](https://github.com/humbug/php-scoper) - outil d’isolation des namespaces PHP
