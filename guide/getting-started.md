# Getting Started

## Installation

### Install from the WordPress Plugin Directory

1. Go to the Plugins page in the WordPress admin.
2. Search for A Ripple Song in the plugin search box.
3. Find the A Ripple Song plugin and click **Install Now**.
4. After installation finishes, click **Activate**.

### Install from a Plugin Package

1. Download the latest `a-ripple-song-podcast.zip` from [GitHub Releases](https://github.com/jiejia/a-ripple-song/releases), or download `a-ripple-song-podcast.zip` from the [WordPress Plugin Directory](https://wordpress.org/plugins/a-ripple-song/).
2. In the WordPress admin, go to **Add Plugins**, click **Upload Plugin**, choose the zip file, and click **Install Now**.
3. Click **Activate**.

### Install from the Command Line

1. Make sure WP-CLI is installed. If it is not installed yet, see <https://wp-cli.org/>.
2. Go to the WordPress root directory and run:

```sh
wp plugin install a-ripple-song
```
