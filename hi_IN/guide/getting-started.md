# शुरुआत करें

## इंस्टॉलेशन
### WordPress प्लगइन डायरेक्टरी से
1. WordPress एडमिन में प्लगइन्स पेज खोलें
2. खोज बॉक्स में A Ripple Song लिखें और Enter दबाएँ
3. जब A Ripple Song प्लगइन दिखे, तो 【अभी इंस्टॉल करें】 पर क्लिक करें
4. इंस्टॉल पूरा होने पर 【सक्रिय करें】 पर क्लिक करें

### प्लगइन पैकेज से
1. [Github Release](https://github.com/jiejia/a-ripple-song/releases) से `a-ripple-song-podcast.zip` का नवीनतम संस्करण डाउनलोड करें, या [WordPress Themes](https://wordpress.org/plugins/a-ripple-song/) से प्राप्त करें
2. WordPress में 【नया प्लगइन जोड़ें】 पेज खोलें, 【प्लगइन अपलोड करें】 पर क्लिक करें, फ़ाइल चुनें और 【अभी इंस्टॉल करें】 दबाएँ
3. 【सक्रिय करें】 पर क्लिक करें

### कमांड लाइन से
1. पहले जाँचें कि wp cli इंस्टॉल है या नहीं; यदि नहीं, तो https://wp-cli.org/ से इंस्टॉल करें
2. WordPress root में यह कमांड चलाएँ: `wp plugin install a-ripple-song`
