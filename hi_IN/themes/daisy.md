# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song) एक क्लासिक WordPress थीम है। [पॉडकास्ट प्लगइन](https://github.com/jiejia/a-ripple-song) इंस्टॉल करने के बाद ऑडियो प्लेयर और पॉडकास्ट widgets इस्तेमाल किए जा सकते हैं।

## इंस्टॉलेशन

### WordPress थीम डायरेक्टरी से
1. WordPress एडमिन में 【नई थीम जोड़ें】 पेज खोलें
2. खोज बॉक्स में Daisy A Ripple Song लिखें और Enter दबाएँ
3. जब Daisy A Ripple Song थीम दिखे, तो 【इंस्टॉल】 पर क्लिक करें
4. इंस्टॉल पूरा होने पर 【सक्रिय करें】 पर क्लिक करें

### थीम पैकेज से
1. [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases) से `daisy-a-ripple-song-podcast.zip` का नवीनतम संस्करण डाउनलोड करें, या [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/) से प्राप्त करें
2. WordPress में 【थीम जोड़ें】 पेज खोलें, 【थीम अपलोड करें】 पर क्लिक करें, फ़ाइल चुनें और 【अभी इंस्टॉल करें】 दबाएँ
3. 【सक्रिय करें】 पर क्लिक करें

### कमांड लाइन से
1. पहले जाँचें कि wp cli इंस्टॉल है या नहीं; यदि नहीं, तो https://wp-cli.org/ से इंस्टॉल करें
2. WordPress root में `wp theme install daisy-a-ripple-song` चलाएँ
