# 安装

本页介绍了如何安装 "A Ripple Song" 主题及其所需的配套插件。

## 系统要求

- **WordPress**: 6.6 或更高版本
- **PHP**: 8.2 或更高版本
- **服务器**: 音频播放功能需要 HTTPS 支持

## 安装步骤

安装过程分为两部分：安装**主题**（负责外观和感觉）和**播客插件**（负责剧集管理）。

### 第一步：安装插件（核心功能）

`a-ripple-song-podcast` 插件处理核心播客功能，包括剧集管理和 RSS Feed 生成。

1. 从 [GitHub Release 页面](https://github.com/jiejia/a-ripple-song-podcast/releases) 下载插件。
2. 进入 WordPress 后台 → **插件** → **安装插件**。
3. 点击 **上传插件** 并选择下载的 zip 文件。
4. 点击 **安装**，然后点击 **启用插件**。

:::info
如果没有这个插件，主题将作为一个标准的博客主题运行，没有播客特定的功能（播放器、剧集列表等）。
:::

### 第二步：安装主题（设计）

1. 从 [GitHub Release 页面](https://github.com/jiejia/a-ripple-song/releases) 下载主题包。
2. 进入 WordPress 后台 → **外观** → **主题**。
3. 点击 **安装主题** → **上传主题**。
4. 选择下载的主题 zip 文件。
5. 点击 **安装**，然后点击 **启用**。

### 第三步：验证安装

在启用主题和插件后，你应该在 WordPress 后台侧边栏看到一个新的 **"Podcasts"**（播客）菜单项。这表明剧集管理系统已激活。

## 升级

### 升级主题
1. 备份你的网站（数据库和文件）。
2. 下载新版本的主题。
3. 上传并替换现有主题（WordPress 会询问是否替换当前版本）。

### 升级插件
1. 禁用并删除旧版本的插件（不用担心，你的数据存储在数据库中）。
2. 按照上述安装步骤安装新版本。

## 卸载

- **主题**：切换到其他主题。
- **插件**：禁用插件。你的剧集数据将保留在数据库中，除非你明确删除了它们（取决于你的清理设置）。
