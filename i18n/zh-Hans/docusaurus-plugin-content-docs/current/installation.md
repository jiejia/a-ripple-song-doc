# 安装

本页说明如何安装主题，以及常见安装问题的处理方式（占位内容）。

## 环境要求（占位）

- WordPress：6.x+
- PHP：8.1+
- 服务器：支持 HTTPS

## 安装方式

### 方式 A：上传主题包

1. 下载主题压缩包（占位：从哪里下载）
2. 后台 → 外观 → 主题 → 上传
3. 启用主题

### 方式 B：从代码仓库部署（可选）

适用于开发者或自定义部署（占位）。

## 升级

- 建议先备份数据库与主题设置
- 升级后清理缓存（CDN/插件/浏览器）

## 卸载

- 切换到其他主题
- 可选：删除主题相关数据（请谨慎）
