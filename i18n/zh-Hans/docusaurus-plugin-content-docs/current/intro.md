# A Ripple Song 播客主题

**A Ripple Song** 是一个现代化的 WordPress 一站式播客解决方案。它采用模块化架构，将设计与功能分离：

- **主题 (`a-ripple-song`)**：负责视觉呈现、播放器界面和前端体验。
- **插件 (`a-ripple-song-podcast`)**：管理数据结构（剧集、RSS Feed）和核心逻辑。

这种设计确保了即使你未来切换主题，你的播客数据依然安全。

## 核心功能

- **专业的播客管理**：专属的“剧集”文章类型，支持分季、集数编号和嘉宾管理。
- **沉浸式音频播放器**：支持页面跳转间持续播放（SPA 体验），并配备可视化波形图。
- **现代技术栈**：基于 TailwindCSS v4、Alpine.js 和 Vite 构建，性能极佳。
- **符合标准**：完全支持 Podcasting 2.0 命名空间和 iTunes 元数据标准。
- **国际化**：内置多语言支持（英文、中文等）。

## 工作原理

1. **安装**：在你的 WordPress 站点上安装插件和主题。
2. **配置**：设置播客详情（封面图、描述、RSS 设置）。
3. **发布**：像写博客文章一样创建新的剧集。
4. **分发**：使用生成的 RSS Feed 将你的节目提交到 Apple Podcasts、Spotify 等平台。
