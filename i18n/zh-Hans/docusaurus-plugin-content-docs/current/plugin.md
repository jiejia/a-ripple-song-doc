# A Ripple Song 播客插件

**A Ripple Song 播客插件** (`a-ripple-song-podcast`) 是你播客网站的引擎。它独立于主题运行，确保你的数据可移植且安全。

## 核心功能

### 1. 剧集管理
该插件注册了一个自定义文章类型 `ars_episode`。这允许你将剧集与常规博客文章分开管理。
- **季数与集数编号**：结构化地组织你的内容。
- **嘉宾管理**：将嘉宾与剧集关联。
- **时长与文件大小**：从你的音频文件中自动提取。

### 2. RSS Feed 生成
符合标准的 RSS Feed 对播客分发至关重要。
- **Feed URL**：通常位于 `/feed/podcast`（或配置的 slug）。
- **iTunes 标签**：包含所有必要的 `itunes:*` 标签（封面、是否包含露骨内容、分类等）。
- **Podcasting 2.0**：支持现代命名空间，如 `podcast:transcript` 和 `podcast:person`。

### 3. 播放器集成
虽然**主题**处理*视觉*播放器，但**插件**提供播放所需的*数据*：
- 音频文件 URL
- MIME 类型
- 时长
- 封面图

## 配置

前往 WordPress 后台的 **设置 → 播客** 进行配置：

- **播客标题与描述**：如果需要，可覆盖站点设置。
- **封面图片**：你的节目的主要艺术作品。
- **所有者信息**：用于 iTunes 验证的姓名和电子邮件。
- **分类**：为目录选择主要和次要分类。
