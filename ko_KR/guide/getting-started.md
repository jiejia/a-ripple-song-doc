# 시작하기

## 설치
### WordPress 플러그인 디렉터리에서
1. WordPress 관리자에서 플러그인 페이지를 엽니다
2. 검색창에 A Ripple Song을 입력하고 Enter를 누릅니다
3. A Ripple Song 플러그인이 보이면 【지금 설치】를 클릭합니다
4. 설치가 끝나면 【활성화】를 클릭합니다

### 플러그인 패키지에서
1. [Github Release](https://github.com/jiejia/a-ripple-song/releases)에서 `a-ripple-song-podcast.zip` 최신 버전을 내려받거나, [WordPress Themes](https://wordpress.org/plugins/a-ripple-song/)에서 받습니다
2. WordPress의 【새 플러그인 추가】 페이지를 열고 【플러그인 업로드】를 클릭한 뒤 파일을 선택하고 【지금 설치】를 누릅니다
3. 【활성화】를 클릭합니다

### 명령줄에서
1. 먼저 wp cli가 설치되어 있는지 확인하고, 없다면 https://wp-cli.org/ 에서 설치합니다
2. WordPress 루트에서 `wp plugin install a-ripple-song` 을 실행합니다
