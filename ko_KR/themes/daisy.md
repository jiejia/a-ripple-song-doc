# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song)은 클래식 WordPress 테마입니다. [팟캐스트 플러그인](https://github.com/jiejia/a-ripple-song)을 설치하면 오디오 플레이어와 팟캐스트 위젯을 사용할 수 있습니다.

## 설치

### WordPress 테마 디렉터리에서
1. WordPress 관리자에서 【새 테마 추가】 페이지를 엽니다
2. 검색창에 Daisy A Ripple Song을 입력하고 Enter를 누릅니다
3. Daisy A Ripple Song 테마가 보이면 【설치】를 클릭합니다
4. 설치가 끝나면 【활성화】를 클릭합니다

### 테마 패키지에서
1. [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases)에서 `daisy-a-ripple-song-podcast.zip` 최신 버전을 내려받거나, [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/)에서 받습니다
2. WordPress의 【테마 추가】 페이지를 열고 【테마 업로드】를 클릭한 뒤 파일을 선택하고 【지금 설치】를 누릅니다
3. 【활성화】를 클릭합니다

### 명령줄에서
1. 먼저 wp cli가 설치되어 있는지 확인하고, 없다면 https://wp-cli.org/ 에서 설치합니다
2. WordPress 루트에서 `wp theme install daisy-a-ripple-song` 을 실행합니다

## 기능

- 56가지의 아름다운 색상 조합
- 국제화 지원
- 데이터 추적
- Swup 기반의 새로고침 없는 페이지 전환
- 테마 스타일에 맞춘 여러 위젯
- 몰입형 오디오 플레이어
