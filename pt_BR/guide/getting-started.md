# Primeiros passos

## Instalação
### Pelo repositório de plugins do WordPress
1. Abra a página de plugins no painel do WordPress
2. Digite A Ripple Song na busca e pressione Enter
3. Quando o plugin A Ripple Song aparecer, clique em 【Instalar agora】
4. Quando a instalação terminar, clique em 【Ativar】

### A partir de um pacote do plugin
1. Baixe a versão mais recente de `a-ripple-song-podcast.zip` em [Github Release](https://github.com/jiejia/a-ripple-song/releases), ou em [WordPress Themes](https://wordpress.org/plugins/a-ripple-song/)
2. Abra a página 【Adicionar plugin】 no WordPress, clique em 【Enviar plugin】, escolha o arquivo e pressione 【Instalar agora】
3. Clique em 【Ativar】

### Instalação via linha de comando
1. Primeiro verifique se o wp cli está instalado; caso contrário, instale em https://wp-cli.org/
2. Na raiz do WordPress, execute `wp plugin install a-ripple-song`
