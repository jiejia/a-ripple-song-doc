# O que é A Ripple Song?

A Ripple Song é um plugin de podcast para WordPress e o núcleo de todo o sistema. Ele gerencia a estrutura de conteúdo, a administração, o tratamento dos metadados de áudio e a publicação RSS, oferecendo ao WordPress uma capacidade completa de publicação de podcasts.

## Capacidades principais

- Adiciona tipos de conteúdo e categorias específicas para podcasts no WordPress
- Fornece campos e ajustes de administração dedicados
- Detecta automaticamente duração, tamanho e formato do áudio
- Gera um RSS compatível com iTunes e Podcasting 2.0, pronto para distribuição nas principais plataformas
- Inclui campos de estatísticas

## Relação com o tema

O plugin A Ripple Song é o núcleo. O tema é a interface de apresentação.

- O plugin cuida das funções, dos dados e da lógica de negócio
- O tema cuida da exibição das páginas e da experiência do usuário

Sem o tema, o plugin continua sendo o núcleo do sistema de podcast. O tema apenas apresenta esse conteúdo de forma mais completa e agradável de navegar.

## Resumo

A Ripple Song é o plugin central que dá ao WordPress capacidades completas de podcasting. O tema transforma esse conteúdo em uma interface de site realmente utilizável.
