# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song) é um tema clássico do WordPress. Depois de instalar o [plugin de podcast](https://github.com/jiejia/a-ripple-song), ele adiciona um player de áudio e widgets de podcast.

[https://podcast.aripplesong.me/](https://podcast.aripplesong.me/)

## Instalação

### Pelo repositório de temas do WordPress
1. Abra a página 【Adicionar tema】 no painel do WordPress
2. Digite Daisy A Ripple Song na busca e pressione Enter
3. Quando o tema Daisy A Ripple Song aparecer, clique em 【Instalar】
4. Quando a instalação terminar, clique em 【Ativar】

### A partir de um pacote do tema
1. Baixe a versão mais recente de `daisy-a-ripple-song-podcast.zip` em [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases), ou em [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/)
2. Abra a página 【Adicionar tema】, clique em 【Enviar tema】，escolha o arquivo e pressione 【Instalar agora】
3. Clique em 【Ativar】

### Instalação via linha de comando
1. Primeiro verifique se o wp cli está instalado; caso contrário, instale em https://wp-cli.org/
2. Na raiz do WordPress, execute `wp theme install daisy-a-ripple-song`

## Características

- 56 combinações de cores elegantes
- Suporte à internacionalização
- Rastreamento de dados
- Transições de página sem recarga baseadas em Swup
- Vários widgets adaptados ao estilo do tema
- Player de áudio imersivo

## Stack técnica
- [Vite](https://github.com/vitejs/vite) - ferramenta de desenvolvimento e build front-end
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) - framework CSS utilitário
- [daisyUI](https://github.com/saadeghi/daisyui) - biblioteca de componentes UI baseada em Tailwind CSS
- [Alpine.js](https://github.com/alpinejs/alpine) - framework leve de interação front-end
- [Lucide](https://github.com/lucide-icons/lucide) - biblioteca de ícones open source
- [Simple Icons](https://github.com/simple-icons/simple-icons) - biblioteca de ícones de marcas
- [Swup](https://github.com/swup/swup) - biblioteca de navegação sem recarga
- [Howler.js](https://github.com/goldfire/howler.js) - biblioteca de reprodução de áudio
- [audioMotion-analyzer](https://github.com/hvianna/audioMotion-analyzer) - biblioteca de visualização espectral de áudio
- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - framework de campos personalizados do WordPress
- [PHP-Scoper](https://github.com/humbug/php-scoper) - ferramenta de isolamento de namespaces PHP
