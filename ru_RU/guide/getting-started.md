# Быстрый старт

## Установка
### Из каталога плагинов WordPress
1. Откройте страницу плагинов в админке WordPress
2. Введите A Ripple Song в строке поиска и нажмите Enter
3. Когда появится плагин A Ripple Song, нажмите 【Установить сейчас】
4. После установки нажмите 【Активировать】

### Из архива плагина
1. Скачайте последнюю версию `a-ripple-song-podcast.zip` из [Github Release](https://github.com/jiejia/a-ripple-song/releases) или из [WordPress Themes](https://wordpress.org/plugins/a-ripple-song/)
2. Откройте страницу 【Добавить плагин】 в WordPress, нажмите 【Загрузить плагин】，выберите файл и нажмите 【Установить сейчас】
3. Нажмите 【Активировать】

### Установка через командную строку
1. Сначала проверьте, установлен ли wp cli; если нет, установите его с https://wp-cli.org/
2. В корне WordPress выполните `wp plugin install a-ripple-song`
