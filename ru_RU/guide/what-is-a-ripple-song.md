# Что такое A Ripple Song?

A Ripple Song — это плагин подкастов для WordPress и ядро всей системы. Он отвечает за структуру контента, администрирование, обработку аудиометаданных и публикацию RSS, чтобы WordPress получил полноценные возможности для подкастинга.

## Основные возможности

- Добавляет для WordPress типы записей и категории для подкастов
- Предоставляет специальные поля и настройки администратора
- Автоматически определяет длительность, размер и формат аудио
- Генерирует RSS, совместимый с iTunes и Podcasting 2.0, готовый к публикации на основных платформах
- Предоставляет поля статистики

## Связь с темой

Плагин A Ripple Song — это ядро. Тема — это слой представления.

- Плагин отвечает за функции, данные и бизнес-логику
- Тема отвечает за отображение страниц и пользовательский опыт

Без темы плагин всё равно остаётся ядром системы подкаста. Тема просто показывает этот контент в более удобном и полном виде.

## Итог

A Ripple Song — это центральный плагин, который даёт WordPress полноценные возможности подкастинга. Тема превращает этот контент в действительно удобный интерфейс сайта.
