# Управление подкастами

## Добавление
1. Откройте страницу добавления подкаста
![Скриншот настроек подкаста](/images/manage-1.png)
2. Заполните данные подкаста, загрузите аудиофайл и изображение, затем нажмите 【Опубликовать】
![Скриншот настроек подкаста](/images/manage-2.png)

## Редактирование
1. Откройте страницу редактирования подкаста
![Скриншот настроек подкаста](/images/manage-3.png)
2. Отредактируйте данные подкаста и нажмите 【Сохранить】
![Скриншот настроек подкаста](/images/manage-4.png)

## Справка по полям

  | Поле | Описание |
  |---|---|
  | Аудиофайл <span style="color: red;">*</span> | Обязательно. Загрузите аудиофайл или укажите URL аудио (https). |
  | Длительность (секунды) | Автоматически определяется при сохранении из “Audio File”, в секундах, только чтение. |
  | Длина аудио (байты) | Автоматически определяется при сохранении из “Audio File”, только чтение. |
  | Тип MIME аудио | Автоматически определяется при сохранении из “Audio File”, только чтение, по умолчанию `audio/mpeg`. |
  | Откровенный контент <span style="color: red;">*</span> | Обязательно. Выберите `clean` или `explicit`. |
  | Тип эпизода <span style="color: red;">*</span> | Обязательно. Выберите `full`, `trailer` или `bonus`. |
  | Номер эпизода | Необязательно, но рекомендуется. Целое число. |
  | Номер сезона | Необязательно. Целое число. |
  | Автор эпизода (переопределение) | Необязательно. Переопределяет автора канала для этого эпизода. |
  | Обложка эпизода (квадратная) | Необязательно. Квадратное изображение, рекомендуется 1400-3000 px. Переопределяет обложку канала. |
  | Транскрипт (необязательно) | Необязательно. Загрузите файл расшифровки или укажите URL, поддерживаются vtt/srt/txt/pdf и др. |
  | Заголовок iTunes (необязательно) | Необязательно. Замещающее значение заголовка эпизода для Apple Podcasts. |
  | Главы (Podcasting 2.0) | Необязательно. JSON-файл или URL глав. |
  | Тип MIME глав | Необязательно. MIME-тип файла глав, по умолчанию `application/json+chapters`. |
  | Звуковые фрагменты (Podcasting 2.0) | Составное поле. Добавьте один или несколько фрагментов; каждый содержит время начала, длительность и заголовок. |
  | Подзаголовок | Необязательно. Короткий подзаголовок iTunes. |
  | Сводка | Необязательно. Текстовое резюме iTunes. |
  | Пользовательский GUID (необязательно) | Необязательно. Пользовательский RSS GUID; если пусто, используется ссылка эпизода. |
  | Блокировка iTunes | Необязательно. `yes` скрывает эпизод в Apple Podcasts, по умолчанию `no`. |
  | Участники | Выберите участников эпизода, обычно администраторов, авторов или редакторов. |
  | Гости | Выберите гостей эпизода, обычно участников. |

## Примечания
- Поля, отмеченные <span style="color: red;">*</span>, обязательны.
