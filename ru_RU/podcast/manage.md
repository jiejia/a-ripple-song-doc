# Управление подкастами

## Добавление
1. Откройте страницу добавления подкаста
![Скриншот настроек подкаста](/images/manage-1.png)
2. Заполните данные подкаста, загрузите аудиофайл и изображение, затем нажмите 【Опубликовать】
![Скриншот настроек подкаста](/images/manage-2.png)

## Редактирование
1. Откройте страницу редактирования подкаста
![Скриншот настроек подкаста](/images/manage-3.png)
2. Отредактируйте данные подкаста и нажмите 【Сохранить】
![Скриншот настроек подкаста](/images/manage-4.png)

## Справка по полям

  | Поле | Описание |
  |---|---|
  | Audio File <span style="color: red;">*</span> | Обязательно. Загрузите аудиофайл или укажите URL аудио (https). |
  | Duration (seconds) | Автоматически определяется при сохранении из “Audio File”, в секундах, только чтение. |
  | Audio Length (bytes) | Автоматически определяется при сохранении из “Audio File”, только чтение. |
  | Audio MIME Type | Автоматически определяется при сохранении из “Audio File”, только чтение, по умолчанию `audio/mpeg`. |
  | Explicit <span style="color: red;">*</span> | Обязательно. Выберите `clean` или `explicit`. |
  | Episode Type <span style="color: red;">*</span> | Обязательно. Выберите `full`, `trailer` или `bonus`. |
  | Episode Number | Необязательно, но рекомендуется. Целое число. |
  | Season Number | Необязательно. Целое число. |
  | Episode Author (override) | Необязательно. Переопределяет автора канала для этого эпизода. |
  | Episode Cover (square) | Необязательно. Квадратное изображение, рекомендуется 1400-3000 px. Переопределяет обложку канала. |
  | Transcript (optional) | Необязательно. Загрузите файл расшифровки или укажите URL, поддерживаются vtt/srt/txt/pdf и др. |
  | iTunes Title (optional) | Необязательно. Замещающее значение заголовка эпизода для Apple Podcasts. |
  | Chapters (Podcasting 2.0) | Необязательно. JSON-файл или URL глав. |
  | Chapters MIME Type | Необязательно. MIME-тип файла глав, по умолчанию `application/json+chapters`. |
  | Soundbites (Podcasting 2.0) | Составное поле. Добавьте один или несколько фрагментов; каждый содержит время начала, длительность и заголовок. |
  | Subtitle | Необязательно. Короткий подзаголовок iTunes. |
  | Summary | Необязательно. Текстовое резюме iTunes. |
  | Custom GUID (optional) | Необязательно. Пользовательский RSS GUID; если пусто, используется ссылка эпизода. |
  | iTunes Block | Необязательно. `yes` скрывает эпизод в Apple Podcasts, по умолчанию `no`. |
  | Members | Выберите участников эпизода, обычно администраторов, авторов или редакторов. |
  | Guests | Выберите гостей эпизода, обычно участников. |

## Примечания
- Поля, отмеченные <span style="color: red;">*</span>, обязательны.
