# Настройки подкаста
Эти поля в основном используются для генерации RSS-потока подкаста.

## Как открыть
![Скриншот настроек подкаста](/images/settings-1.png)

## Справка по полям
| Поле | Описание |
|---|---|
| RSS URL подкаста | Показывает текущий RSS-адрес подкаста только для чтения. Можно нажать в поле, чтобы выделить и скопировать его, либо открыть сам feed. |
| Название подкаста <span style="color: red;">*</span> | Обязательно. Название подкаста. Если пусто, RSS вернётся к названию сайта. |
| Подзаголовок подкаста | Необязательно. Краткий подзаголовок или слоган подкаста, который показывают некоторые приложения. |
| Описание подкаста <span style="color: red;">*</span> | Обязательно. Описание подкаста в простом тексте. Попадает в RSS как description / сводка iTunes. |
| Автор подкаста (itunes:author) <span style="color: red;">*</span> | Обязательно. Автор или продюсер подкаста, который отображается в каталогах. |
| Имя владельца <span style="color: red;">*</span> | Обязательно. Значение `<itunes:owner><itunes:name>` для сведений о владельце. |
| Email владельца <span style="color: red;">*</span> | Обязательно. Значение `<itunes:owner><itunes:email>`. Укажите адрес, который может получать письма с проверкой. |
| Обложка подкаста (1400-3000px квадратная) <span style="color: red;">*</span> | Обязательно. Обложка подкаста. Формат JPG/PNG, квадратная, 1400-3000 px и меньше 512 KB. |
| Флаг откровенного контента по умолчанию <span style="color: red;">*</span> | Обязательно. Флаг откровенного контента по умолчанию. Выберите clean или explicit. Поля эпизода могут его переопределять. |
| Язык (RFC 5646) <span style="color: red;">*</span> | Обязательно. Язык подкаста, например en-US, zh-CN или ja-JP. |
| Основная категория (Apple Podcasts) <span style="color: red;">*</span> | Обязательно. Основная категория Apple Podcasts. Нужно выбрать хотя бы одну. |
| Вторичная категория (необязательно) | Необязательно. Вторая категория, которую поддерживают некоторые каталоги подкастов. |
| Авторские права (необязательно) | Необязательно. Уведомление об авторских правах, которое выводится в `<copyright>`. |
| Тип iTunes (itunes:type) | Необязательно. Тип шоу Apple Podcasts. Варианты: не задано, episodic, serial. |
| Заголовок iTunes (необязательно) | Необязательно. Специальное название для Apple Podcasts. Используйте только если оно должно отличаться от обычного названия подкаста. |
| Блокировка iTunes (itunes:block) | Необязательно. yes скрывает весь подкаст в Apple Podcasts. Значение по умолчанию: no. |
| Завершённость iTunes (itunes:complete) | Необязательно. yes означает, что подкаст завершён и больше не будет обновляться. Значение по умолчанию: no. |
| Новый URL RSS iTunes (itunes:new-feed-url) | Необязательно. Используйте только при переносе на новый RSS-адрес. |
| podcast:locked | Поле Podcasting 2.0. По умолчанию yes, чтобы предотвратить несанкционированный перенос feed. |
| владелец podcast:locked (необязательно) | Необязательно. E-mail владельца, используемый для проверки переноса feed в Podcasting 2.0. |
| podcast:guid (необязательно) | Необязательно. GUID Podcasting 2.0. Если пусто, feed использует URL сайта как запасной вариант. |
| Код проверки Apple Podcasts | Необязательно. Код проверки Apple Podcasts, выводится как `podcast:txt purpose="applepodcastsverify"`. |
| Ссылки финансирования Podcasting 2.0 | Составное поле. Можно добавить несколько ссылок на финансирование или поддержку; каждая запись содержит URL и Label. Если пусто, `podcast:funding` не выводится. |
| Генератор (необязательно) | Необязательно. Содержимое `<generator>` RSS. Если пусто, тег не выводится. |

## Примечания
- Поля, отмеченные <span style="color: red;">*</span>, обязательны.
