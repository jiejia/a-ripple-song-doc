# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song) — это классическая тема WordPress. После установки [плагина подкаста](https://github.com/jiejia/a-ripple-song) она добавляет аудиоплеер и виджеты подкастов.

## Установка

### Из каталога тем WordPress
1. Откройте страницу 【Добавить тему】 в админке WordPress
2. Введите Daisy A Ripple Song в поиске и нажмите Enter
3. Когда появится тема Daisy A Ripple Song, нажмите 【Установить】
4. После установки нажмите 【Активировать】

### Из архива темы
1. Скачайте последнюю версию `daisy-a-ripple-song-podcast.zip` из [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases) или из [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/)
2. Откройте страницу 【Добавить тему】, нажмите 【Загрузить тему】, выберите файл и нажмите 【Установить сейчас】
3. Нажмите 【Активировать】

### Установка через командную строку
1. Сначала проверьте, установлен ли wp cli; если нет, установите его с https://wp-cli.org/
2. В корне WordPress выполните `wp theme install daisy-a-ripple-song`

## Особенности

- 56 цветовых схем
- Поддержка интернационализации
- Отслеживание данных
- Переходы страниц без перезагрузки на базе Swup
- Несколько виджетов, адаптированных под стиль темы
- Иммерсивный аудиоплеер
