# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song) is a classic WordPress theme with a built-in audio player, podcast widgets, and related features.

## Installation

### Install from the WordPress Theme Directory

1. Go to **Add Themes** in the WordPress admin.
2. Search for Daisy A Ripple Song in the theme search box.
3. Find the Daisy A Ripple Song theme and click **Install**.
4. After installation finishes, click **Activate**.

### Install from a Theme Package

1. Download the latest `daisy-a-ripple-song-podcast.zip` from [GitHub Releases](https://github.com/jiejia/daisy-a-ripple-song/releases), or download `daisy-a-ripple-song-podcast.zip` from the [WordPress Theme Directory](https://wordpress.org/themes/daisy-a-ripple-song/).
2. In the WordPress admin, go to **Add Themes**, click **Upload Theme**, choose the zip file, and click **Install Now**.
3. Click **Activate**.

### Install from the Command Line

1. Make sure WP-CLI is installed. If it is not installed yet, see <https://wp-cli.org/>.
2. Go to the WordPress root directory and run:

```sh
wp theme install daisy-a-ripple-song
```

## Features

- 56 polished theme color schemes, including 26 light themes and 26 dark themes.
- Internationalization support.
- Data tracking.
- Swup-based page transitions without full page reloads.
- Multiple widgets adapted to the theme style. Some features require the podcast plugin.
- Immersive audio player. Requires the podcast plugin.

## Tech Stack

- [Vite](https://github.com/vitejs/vite) - Frontend development and build tool.
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) - Utility-first CSS framework.
- [daisyUI](https://github.com/saadeghi/daisyui) - Tailwind CSS component library.
- [Alpine.js](https://github.com/alpinejs/alpine) - Lightweight frontend interaction framework.
- [Lucide](https://github.com/lucide-icons/lucide) - Open-source icon library.
- [Simple Icons](https://github.com/simple-icons/simple-icons) - Brand icon library.
- [Swup](https://github.com/swup/swup) - Page transition library without full page reloads.
- [Howler.js](https://github.com/goldfire/howler.js) - Audio playback library.
- [audioMotion-analyzer](https://github.com/hvianna/audioMotion-analyzer) - Audio spectrum visualization library.
- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - WordPress custom field framework.
- [PHP-Scoper](https://github.com/humbug/php-scoper) - PHP namespace isolation tool.
