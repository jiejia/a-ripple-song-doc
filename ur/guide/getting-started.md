# آغاز کیسے کریں

## انسٹالیشن
### WordPress پلگ ان ڈائریکٹری سے
1. WordPress ایڈمن میں پلگ انز کا صفحہ کھولیں
2. سرچ باکس میں A Ripple Song لکھیں اور Enter دبائیں
3. جب A Ripple Song پلگ ان دکھائی دے تو 【ابھی انسٹال کریں】 پر کلک کریں
4. انسٹال مکمل ہونے پر 【فعال کریں】 پر کلک کریں

### پلگ ان پیکیج سے
1. [Github Release](https://github.com/jiejia/a-ripple-song/releases) سے `a-ripple-song-podcast.zip` کا تازہ ترین ورژن ڈاؤن لوڈ کریں، یا [WordPress Themes](https://wordpress.org/plugins/a-ripple-song/) سے حاصل کریں
2. WordPress میں 【نیا پلگ ان شامل کریں】 کھولیں، 【پلگ ان اپ لوڈ کریں】 پر کلک کریں، فائل منتخب کریں اور 【ابھی انسٹال کریں】 دبائیں
3. 【فعال کریں】 پر کلک کریں

### کمانڈ لائن سے
1. پہلے چیک کریں کہ wp cli انسٹال ہے یا نہیں؛ اگر نہیں تو https://wp-cli.org/ سے انسٹال کریں
2. WordPress روٹ میں یہ کمانڈ چلائیں: `wp plugin install a-ripple-song`
