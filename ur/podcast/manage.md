# پوڈکاسٹ کا انتظام

## شامل کرنا
1. پوڈکاسٹ شامل کرنے والا صفحہ کھولیں
![پوڈکاسٹ سیٹنگز اسکرین شاٹ](/images/manage-1.png)
2. پوڈکاسٹ کی معلومات بھر دیں، آڈیو فائل اور تصویر اپ لوڈ کریں، پھر 【شائع کریں】 پر کلک کریں
![پوڈکاسٹ سیٹنگز اسکرین شاٹ](/images/manage-2.png)

## ترمیم
1. پوڈکاسٹ ترمیم والا صفحہ کھولیں
![پوڈکاسٹ سیٹنگز اسکرین شاٹ](/images/manage-3.png)
2. معلومات میں ترمیم کریں اور 【محفوظ کریں】 پر کلک کریں
![پوڈکاسٹ سیٹنگز اسکرین شاٹ](/images/manage-4.png)

## فیلڈ حوالہ

  | فیلڈ | وضاحت |
  |---|---|
  | Audio File <span style="color: red;">*</span> | لازمی۔ آڈیو فائل اپ لوڈ کریں یا آڈیو URL (https) دیں۔ |
  | Duration (seconds) | محفوظ کرنے پر “Audio File” سے خود بخود معلوم ہوتا ہے، سیکنڈز میں، صرف پڑھنے کے لیے۔ |
  | Audio Length (bytes) | محفوظ کرنے پر “Audio File” سے خود بخود معلوم ہوتا ہے، صرف پڑھنے کے لیے۔ |
  | Audio MIME Type | محفوظ کرنے پر “Audio File” سے خود بخود معلوم ہوتا ہے، صرف پڑھنے کے لیے، ڈیفالٹ `audio/mpeg`۔ |
  | Explicit <span style="color: red;">*</span> | لازمی۔ `clean` یا `explicit` منتخب کریں۔ |
  | Episode Type <span style="color: red;">*</span> | لازمی۔ `full`، `trailer` یا `bonus` منتخب کریں۔ |
  | Episode Number | اختیاری، مگر تجویز کردہ۔ مکمل عدد۔ |
  | Season Number | اختیاری۔ مکمل عدد۔ |
  | Episode Author (override) | اختیاری۔ اس ایپی سوڈ کے لیے چینل کے مصنف کو بدل دیتا ہے۔ |
  | Episode Cover (square) | اختیاری۔ مربع تصویر، 1400-3000px تجویز ہے۔ چینل کور کو بدل دیتی ہے۔ |
  | Transcript (optional) | اختیاری۔ ٹرانسکرپٹ فائل اپ لوڈ کریں یا URL دیں، vtt/srt/txt/pdf وغیرہ سپورٹ ہیں۔ |
  | iTunes Title (optional) | اختیاری۔ Apple Podcasts کے لیے ایپی سوڈ عنوان کا متبادل۔ |
  | Chapters (Podcasting 2.0) | اختیاری۔ chapters JSON فائل یا URL۔ |
  | Chapters MIME Type | اختیاری۔ chapters فائل کا MIME type، ڈیفالٹ `application/json+chapters`۔ |
  | Soundbites (Podcasting 2.0) | مرکب فیلڈ۔ ایک یا زیادہ highlight clips شامل کریں؛ ہر clip میں start time، duration اور title ہوتا ہے۔ |
  | Subtitle | اختیاری۔ iTunes کا مختصر سب ٹائٹل۔ |
  | Summary | اختیاری۔ iTunes کا سادہ متن والا خلاصہ۔ |
  | Custom GUID (optional) | اختیاری۔ کسٹم RSS GUID؛ خالی ہو تو ایپی سوڈ لنک استعمال ہوتا ہے۔ |
  | iTunes Block | اختیاری۔ `yes` Apple Podcasts میں ایپی سوڈ چھپا دیتا ہے، ڈیفالٹ `no`۔ |
  | Members | ایپی سوڈ کے ممبرز منتخب کریں، عام طور پر ایڈمن، مصنف یا ایڈیٹر۔ |
  | Guests | ایپی سوڈ کے گیسٹس منتخب کریں، عام طور پر کنٹریبیوٹرز۔ |

## نوٹس
- جن فیلڈز پر <span style="color: red;">*</span> ہے وہ لازمی ہیں۔
