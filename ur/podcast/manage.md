# پوڈکاسٹ کا انتظام

## شامل کرنا
1. پوڈکاسٹ شامل کرنے والا صفحہ کھولیں
![پوڈکاسٹ سیٹنگز اسکرین شاٹ](/images/manage-1.png)
2. پوڈکاسٹ کی معلومات بھر دیں، آڈیو فائل اور تصویر اپ لوڈ کریں، پھر 【شائع کریں】 پر کلک کریں
![پوڈکاسٹ سیٹنگز اسکرین شاٹ](/images/manage-2.png)

## ترمیم
1. پوڈکاسٹ ترمیم والا صفحہ کھولیں
![پوڈکاسٹ سیٹنگز اسکرین شاٹ](/images/manage-3.png)
2. معلومات میں ترمیم کریں اور 【محفوظ کریں】 پر کلک کریں
![پوڈکاسٹ سیٹنگز اسکرین شاٹ](/images/manage-4.png)

## فیلڈ حوالہ

  | فیلڈ | وضاحت |
  |---|---|
  | آڈیو فائل <span style="color: red;">*</span> | لازمی۔ آڈیو فائل اپ لوڈ کریں یا آڈیو URL (https) دیں۔ |
  | دورانیہ (سیکنڈز) | محفوظ کرنے پر “Audio File” سے خود بخود معلوم ہوتا ہے، سیکنڈز میں، صرف پڑھنے کے لیے۔ |
  | آڈیو لمبائی (بائٹس) | محفوظ کرنے پر “Audio File” سے خود بخود معلوم ہوتا ہے، صرف پڑھنے کے لیے۔ |
  | آڈیو MIME قسم | محفوظ کرنے پر “Audio File” سے خود بخود معلوم ہوتا ہے، صرف پڑھنے کے لیے، ڈیفالٹ `audio/mpeg`۔ |
  | واضح <span style="color: red;">*</span> | لازمی۔ `clean` یا `explicit` منتخب کریں۔ |
  | ایپی سوڈ کی قسم <span style="color: red;">*</span> | لازمی۔ `full`، `trailer` یا `bonus` منتخب کریں۔ |
  | ایپی سوڈ نمبر | اختیاری، مگر تجویز کردہ۔ مکمل عدد۔ |
  | سیزن نمبر | اختیاری۔ مکمل عدد۔ |
  | ایپی سوڈ مصنف (اوور رائیڈ) | اختیاری۔ اس ایپی سوڈ کے لیے چینل کے مصنف کو بدل دیتا ہے۔ |
  | ایپی سوڈ سرورق (مربع) | اختیاری۔ مربع تصویر، 1400-3000px تجویز ہے۔ چینل کور کو بدل دیتی ہے۔ |
  | ٹرانسکرپٹ (اختیاری) | اختیاری۔ ٹرانسکرپٹ فائل اپ لوڈ کریں یا URL دیں، vtt/srt/txt/pdf وغیرہ سپورٹ ہیں۔ |
  | iTunes عنوان (اختیاری) | اختیاری۔ Apple Podcasts کے لیے ایپی سوڈ عنوان کا متبادل۔ |
  | ابواب (Podcasting 2.0) | اختیاری۔ chapters JSON فائل یا URL۔ |
  | ابواب MIME قسم | اختیاری۔ chapters فائل کا MIME type، ڈیفالٹ `application/json+chapters`۔ |
  | آڈیو جھلکیاں (Podcasting 2.0) | مرکب فیلڈ۔ ایک یا زیادہ highlight clips شامل کریں؛ ہر clip میں start time، duration اور title ہوتا ہے۔ |
  | ذیلی عنوان | اختیاری۔ iTunes کا مختصر سب ٹائٹل۔ |
  | خلاصہ | اختیاری۔ iTunes کا سادہ متن والا خلاصہ۔ |
  | حسبِ ضرورت GUID (اختیاری) | اختیاری۔ کسٹم RSS GUID؛ خالی ہو تو ایپی سوڈ لنک استعمال ہوتا ہے۔ |
  | iTunes بلاک | اختیاری۔ `yes` Apple Podcasts میں ایپی سوڈ چھپا دیتا ہے، ڈیفالٹ `no`۔ |
  | ارکان | ایپی سوڈ کے ممبرز منتخب کریں، عام طور پر ایڈمن، مصنف یا ایڈیٹر۔ |
  | مہمان | ایپی سوڈ کے گیسٹس منتخب کریں، عام طور پر کنٹریبیوٹرز۔ |

## نوٹس
