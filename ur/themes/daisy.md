# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song) ایک کلاسک WordPress تھیم ہے۔ [پوڈکاسٹ پلگ ان](https://github.com/jiejia/a-ripple-song) انسٹال کرنے کے بعد آڈیو پلیئر اور پوڈکاسٹ widgets استعمال کیے جا سکتے ہیں۔

## انسٹالیشن

### WordPress تھیم ڈائریکٹری سے
1. WordPress ایڈمن میں 【نیا تھیم شامل کریں】 والا صفحہ کھولیں
2. سرچ باکس میں Daisy A Ripple Song لکھیں اور Enter دبائیں
3. جب Daisy A Ripple Song تھیم نظر آئے تو 【انسٹال】 پر کلک کریں
4. انسٹال مکمل ہونے پر 【فعال کریں】 پر کلک کریں

### تھیم پیکیج سے
1. [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases) سے `daisy-a-ripple-song-podcast.zip` کا تازہ ترین ورژن ڈاؤن لوڈ کریں، یا [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/) سے حاصل کریں
2. WordPress میں 【تھیم شامل کریں】 کھولیں، 【تھیم اپ لوڈ کریں】 پر کلک کریں، فائل منتخب کریں اور 【ابھی انسٹال کریں】 دبائیں
3. 【فعال کریں】 پر کلک کریں

### کمانڈ لائن سے
1. پہلے چیک کریں کہ wp cli انسٹال ہے یا نہیں؛ اگر نہیں تو https://wp-cli.org/ سے انسٹال کریں
2. WordPress روٹ میں `wp theme install daisy-a-ripple-song` چلائیں
