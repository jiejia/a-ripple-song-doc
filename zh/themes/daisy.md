# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song)  是一个经典 WordPress 主题，自带音频播放器和播客 widget 等功能。

<img width="" src="/images/daisy.png"/>

## 安装

### WordPess 主题市场安装
1. 进入 WordPress 后台【添加主题】页面
2. 在主题搜索框输入 Daisy A Ripple Song 后点击回车键搜索
3. 找到 Daisy A Ripple Song 插件后，点击【安装】
4. 安装完成后，点击【启用】

### 主题包安装
1. 从 [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases) 下载最新版本的 `daisy-a-ripple-song-podcast.zip` 文件到本地，或者从 [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/) 下载`daisy-a-ripple-song-podcast.zip` 到本地
2. 进入 WordPress 后台【添加主题】页面，点击【上传主题】，选择文件后点击【立即安装】
3. 点击【启用】

### 命令行安装
1. 首先检查环境是否安装wp cli，若没有请先安装 https://wp-cli.org/
2. 进入 WordPress 根目录，执行命令 `wp theme install daisy-a-ripple-song`

## 特性

- 56套精美主题配色(26套亮色主题和26套暗色主题)
- 国际化支持
- 数据追踪
- 基于 Swup 的无刷新页面转换技术
- 多个适配主题风格的 widgets(部分功能需安装播客插件)
- 沉浸式音频播放器(需安装播客插件)
