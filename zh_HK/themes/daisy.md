# Daisy A Ripple Song

[Daisy A Ripple Song](https://github.com/jiejia/daisy-a-ripple-song) 是一個經典 WordPress 主題，安裝[播客外掛](https://github.com/jiejia/a-ripple-song)後可使用音訊播放器和播客 widget 等功能。

[https://podcast.aripplesong.me/](https://podcast.aripplesong.me/)

## 安裝

### WordPress 主題市集安裝
1. 進入 WordPress 後台【新增主題】頁面
2. 在主題搜尋框輸入 Daisy A Ripple Song 後按下 Enter 搜尋
3. 找到 Daisy A Ripple Song 主題後，點擊【安裝】
4. 安裝完成後，點擊【啟用】

### 主題封包安裝
1. 從 [Github Release](https://github.com/jiejia/daisy-a-ripple-song/releases) 下載最新版本的 `daisy-a-ripple-song-podcast.zip` 檔案到本機，或從 [WordPress Themes](https://wordpress.org/themes/daisy-a-ripple-song/) 下載 `daisy-a-ripple-song-podcast.zip` 到本機
2. 進入 WordPress 後台【新增主題】頁面，點擊【上傳主題】，選擇檔案後點擊【立即安裝】
3. 點擊【啟用】

### 命令列安裝
1. 先檢查環境是否已安裝 wp cli，若沒有請先安裝 https://wp-cli.org/
2. 進入 WordPress 根目錄，執行指令 `wp theme install daisy-a-ripple-song`

## 特性

- 56 套精美主題配色（26 套亮色主題和 26 套暗色主題）
- 國際化支援
- 資料追蹤
- 基於 Swup 的無刷新頁面轉換技術
- 多個適配主題風格的 widgets（部分功能需安裝播客外掛）
- 沉浸式音訊播放器（需安裝播客外掛）
