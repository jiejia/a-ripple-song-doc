# 快速開始

## 安裝
### 從 WordPress 外掛市集安裝
1. 進入 WordPress 後台外掛頁面
2. 在外掛搜尋框輸入 A Ripple Song 後按下 Enter 搜尋
3. 找到 A Ripple Song 外掛後，點擊【立即安裝】
4. 安裝完成後，點擊【啟用】

### 從外掛封包安裝
1. 從 [Github Release](https://github.com/jiejia/a-ripple-song/releases) 下載最新版本的 `a-ripple-song-podcast.zip` 檔案到本機，或從 [WordPress Themes](https://wordpress.org/plugins/a-ripple-song/) 下載 `a-ripple-song-podcast.zip` 到本機
2. 進入 WordPress 後台【新增外掛】頁面，點擊【上傳外掛】，選擇檔案後點擊【立即安裝】
3. 點擊【啟用】

### 從命令列安裝
1. 先檢查環境是否已安裝 wp cli，若沒有請先安裝 https://wp-cli.org/
2. 進入 WordPress 根目錄，執行指令 `wp plugin install a-ripple-song`
